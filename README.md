## Descripción
E-commerce desarrollado con React utilizando componentes funcionales,
hooks, manejo de estado y navegación entre vistas. Integra Firebase
como base de datos en tiempo real para la gestión de productos y
autenticación de usuarios.

## Nota
Para visualizar el proyecto completo con datos se requiere
configuración de Firebase. El código fuente refleja la estructura
completa de la aplicación.
